```
├── index.ts

src/app/(frontend)/

├── interviews/_components/InterviewCard.tsx
├── grants/_components/GrantCard.tsx
└── news/_components/NewsItem.tsx
```

### BaseCard コンポーネント群
新規カード作成時に使える共通部品：

| コンポーネント | 用途 |
|--------------|------|
| `BaseCard` | カードのラッパー（ホバーアニメーション付き） |
| `CardImage` | 画像エリア（aspect ratio対応） |
| `CardContent` | コンテンツエリア |
| `CardTitle` | タイトル |
| `CardDescription` | 説明文 |
| `CardBadge` | バッジ（variant: default/primary/secondary/accent） |

### 使用例
```tsx
import { BaseCard, CardImage, CardContent, CardTitle, CardBadge } from '@/components/common'

<BaseCard href="/path/to/detail">
  <CardImage src={imageUrl} alt="説明" fallbackIcon={Building2} />
  <CardContent>
    <CardBadge variant="primary">カテゴリ</CardBadge>
    <CardTitle>タイトル</CardTitle>
  </CardContent>
</BaseCard>
```

## 備考
- 既存の個別カードは各ページに最適化されているため、そのまま維持
- BaseCardは新規カード作成時の共通部品として提供
