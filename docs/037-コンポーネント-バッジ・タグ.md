## 実装詳細

### ファイル構成
```
src/components/common/
├── index.ts
└── Badges.tsx    # バッジコンポーネント群
```

### コンポーネント一覧

| コンポーネント | 用途 | カラー |
|--------------|------|--------|
| `CategoryBadge` | 活動分野 | Primary（赤） |
| `AreaBadge` | 活動エリア | Green（緑） |
| `TagBadge` | タグ | Blue（青） |
| `StatusBadge` | ステータス | バリアントで指定 |
| `DeadlineBadge` | 締切状態 | 日数に応じて自動選択 |
| `BadgeList` | 複数バッジのラッパー | - |
| `NewBadge` | 新着 | Orange（オレンジ） |
| `PickupBadge` | ピックアップ | Red（赤） |

### StatusBadge バリアント

| variant | 用途 | カラー |
|---------|------|--------|
| `open` | 受付中 | Green |
| `closed` | 終了 | Muted |
| `soon` | もうすぐ締切 | Orange |
| `urgent` | 締切間近 | Red |
| `info` | 情報 | Blue |

### サイズ

| サイズ | 用途 |
|-------|------|
| `sm` | カード内など小さい表示 |
| `md` | 標準（デフォルト） |
| `lg` | 目立たせたい場所 |

### 使用例

```tsx
import {
  CategoryBadge,
  AreaBadge,
  TagBadge,
  StatusBadge,
  DeadlineBadge,
  BadgeList,
  NewBadge,
} from '@/components/common'

// カテゴリバッジ（アイコン付き）
<CategoryBadge icon>子ども</CategoryBadge>

// エリアバッジ
<AreaBadge>上郷</AreaBadge>

// ステータスバッジ
<StatusBadge variant="open">受付中</StatusBadge>

// 締切バッジ（日数から自動判定）
<DeadlineBadge daysRemaining={5} />  // → 「残り5日」（urgent）
<DeadlineBadge daysRemaining={10} /> // → 「残り10日」（soon）
<DeadlineBadge daysRemaining={30} /> // → 「受付中」（open）
<DeadlineBadge daysRemaining={-1} /> // → 「募集終了」（closed）

// バッジリスト
<BadgeList>
  <CategoryBadge>福祉</CategoryBadge>
  <AreaBadge>市内全域</AreaBadge>
</BadgeList>
```
