# 009 管理画面 - 共通レイアウト

## 概要
管理画面の共通レイアウト（サイドバー、ヘッダー、ナビゲーション）を実装する。

## 関連要件
- 使いやすい管理画面UI
- レスポンシブ対応（タブレット以上推奨）

## 技術仕様

### 使用技術
- shadcn/ui コンポーネント
- Lucide Icons
- Tailwind CSS

### レイアウト構成
```
┌─────────────────────────────────────────┐
│ ヘッダー（ロゴ、ユーザーメニュー）        │
├──────────┬──────────────────────────────┤
│          │                              │
│ サイド   │      メインコンテンツ         │
│ バー     │                              │
│          │                              │
│ - 団体   │                              │
│ - 記事   │                              │
│ - 助成金 │                              │
│ - FAQ    │                              │
│          │                              │
└──────────┴──────────────────────────────┘
```

### ナビゲーション項目
```typescript
const navItems = [
  { label: 'ダッシュボード', href: '/admin', icon: Home },
  { label: '団体管理', href: '/admin/organizations', icon: Users },
  { label: 'インタビュー', href: '/admin/interviews', icon: MessageSquare },
  { label: '助成金', href: '/admin/grants', icon: Coins },
  { label: 'お知らせ', href: '/admin/news', icon: Newspaper },
  { label: 'FAQ', href: '/admin/faqs', icon: HelpCircle },
  { label: 'マスター管理', href: '/admin/master', icon: Settings },
]
```

### コンポーネント構成
```
src/components/admin/
├─ AdminLayout.tsx      # 全体レイアウト
├─ AdminSidebar.tsx     # サイドバー
├─ AdminHeader.tsx      # ヘッダー
├─ AdminNav.tsx         # ナビゲーション
└─ UserMenu.tsx         # ユーザーメニュー（ログアウト等）
```
