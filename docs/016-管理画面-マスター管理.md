# 016 管理画面 - マスター管理

## 概要
マスターデータ（活動分野、活動エリア、タグ）の管理機能を実装する。

## 関連要件
- マスターデータの追加・編集・削除
- 使用中のマスターは削除不可
- 表示順の管理

## 技術仕様

### ページ構成
```
/admin/master                    # マスター管理トップ
/admin/master/categories         # 活動分野
/admin/master/areas              # 活動エリア
/admin/master/tags               # タグ
```

### 各マスターのフォーム項目

#### 活動分野（activity_categories）
```typescript
name: string        // 分野名（必須）
slug: string        // スラッグ
sort_order: number  // 表示順
```

#### 活動エリア（activity_areas）
```typescript
name: string        // エリア名（必須）
slug: string        // スラッグ
sort_order: number  // 表示順
```

#### タグ（tags）
```typescript
name: string        // タグ名（必須）
slug: string        // スラッグ
```

### 共通機能
- 一覧表示
- インライン編集 or モーダル編集
- 使用件数の表示
- 使用中は削除不可（警告表示）

### コンポーネント構成
```
src/app/(admin)/admin/master/
├─ page.tsx                    # マスター管理トップ
├─ categories/page.tsx         # 活動分野
├─ areas/page.tsx              # 活動エリア
├─ tags/page.tsx               # タグ
└─ _components/
    └─ MasterList.tsx          # 汎用マスター一覧
```
