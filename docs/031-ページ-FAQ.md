## 実装詳細

### ファイル構成
```
src/app/(frontend)/faq/
├── page.tsx              # FAQページ（Server Component）
├── loading.tsx           # ローディングUI
└── _components/
    ├── index.ts          # エクスポート
    └── FAQAccordion.tsx  # アコーディオン（Client Component）
```

### 機能
- Q/Aバッジ付きのアコーディオンUI
- 開いた時にボーダー色とシャドウが変化
- 回答はリッチテキスト（HTML）対応
- Framer Motionでスタガードアニメーション
- 空状態のメッセージ表示

## 備考
- アクセシビリティ対応（キーボード操作）はshadcn/uiで担保
- 回答のHTMLはTailwind Typographyでスタイリング
