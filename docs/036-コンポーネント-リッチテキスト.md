
### コンポーネント

#### RichTextRenderer
汎用リッチテキストレンダラー

| Props | 型 | 必須 | 説明 |
|-------|-----|------|------|
| `html` | `string` | ○ | 表示するHTML |
| `size` | `'small' \| 'default' \| 'large'` | - | テキストサイズ（デフォルト: default） |
| `addHeadingIds` | `boolean` | - | 見出しにIDを付与するか（デフォルト: false） |
| `className` | `string` | - | 追加のCSSクラス |

#### ArticleBody
記事本文用のラッパー（見出しID自動付与あり）

```tsx
<ArticleBody html={article.body} />
```

### 使用例

```tsx
import { RichTextRenderer, ArticleBody } from '@/components/common'

// 基本的な使用
<RichTextRenderer html={content} />

// サイズ指定
<RichTextRenderer html={content} size="small" />

// 記事本文（目次連携用）
<ArticleBody html={article.body} />
```

### スタイリング対応
- **見出し**: h2, h3, h4（境界線・マージン付き）
- **段落**: 読みやすい行間
- **リンク**: プライマリカラー、ホバーで下線
- **リスト**: ul/ol（インデント付き）
- **引用**: 左ボーダー + 背景色
- **テーブル**: ボーダー + ヘッダー背景
- **コード**: インラインコード + コードブロック
- **画像**: 角丸
- **区切り線**: hr
